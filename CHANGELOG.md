# 更新日志

## v2.2.0 (2026-05-09)

### 修复
- **API Key 保存失效** - 将存储方案从 `flutter_secure_storage` 替换为 `shared_preferences`，修复 Web 平台密钥重启后丢失的问题
- **设置页密钥显示空白** - 修复因配置异步加载未完成就读取值导致输入框为空的问题，改为等待初始化完成后再填充
- **阅读报告 Y 轴刻度重叠** - 调整 Y 轴最大值为 10 的整数倍，刻度间隔从 5 改为 10，避免数字堆叠
- **阅读报告 X 轴标签重叠** - 图书分类横坐标改为两字缩略显示，并自动处理重复标签

### 优化
- 图书分类柱状图高度增加，标签空间更充裕
- 新增 `shared_preferences` 依赖，提升 Web 平台存储兼容性

## v2.1.0

### 功能
- AI 摘要 - 使用 AI 为书籍生成摘要
- 笔记导出 - 将划线和笔记导出为 Markdown 格式
- 金句卡片 - 将划线生成精美卡片，支持导出高清 PNG
- 阅读报告 - 阅读数据可视化和统计
- Cookie 登录 - 导入微信读书 Cookie 获取数据
